FYP Hashtag Affiliate ("kami", "aplikasi", atau "layanan") menghormati privasi pengguna kami. Privacy Policy ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi FYP Hashtag Affiliate.

1. Informasi yang Kami Kumpulkan

Aplikasi FYP Hashtag Affiliate dirancang dengan fokus pada privasi. Kami mengumpulkan informasi minimal yang diperlukan untuk menyediakan layanan:

  • Data Aplikasi Lokal: Hashtag yang Anda generate, favorit, dan koleksi disimpan secara lokal di perangkat Anda menggunakan SharedPreferences. Data ini tidak dikirim ke server kami.
  • Data untuk AI Generation: Ketika Anda menggunakan fitur AI untuk generate hashtag, input Anda (nama produk, kategori, dll) dikirim ke Google Gemini API untuk diproses. Data ini tidak disimpan oleh kami dan hanya digunakan untuk generate hasil.
  • Data AdMob: Kami menggunakan Google AdMob untuk menampilkan iklan, yang dapat mengumpulkan data anonim untuk personalisasi iklan sesuai dengan kebijakan Google AdMob.
  • Data Perangkat: Informasi dasar perangkat (seperti jenis perangkat, sistem operasi) dapat dikumpulkan untuk keperluan kompatibilitas dan optimasi aplikasi.

2. Penggunaan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Menyediakan dan meningkatkan layanan aplikasi
  • Menampilkan iklan yang relevan melalui Google AdMob
  • Menganalisis penggunaan aplikasi untuk perbaikan fitur
  • Memastikan keamanan dan mencegah penyalahgunaan

3. Penyimpanan Data

Data yang Anda buat di aplikasi (hashtag, favorit, koleksi) disimpan secara lokal di perangkat Anda. Kami tidak menyimpan data ini di server kami. Data akan tetap ada di perangkat Anda sampai Anda menghapus aplikasi atau menghapus data aplikasi secara manual.

4. Iklan dan Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga berikut:

  • Google AdMob: Untuk menampilkan iklan. Google AdMob dapat mengumpulkan informasi untuk menampilkan iklan yang relevan. Untuk informasi lebih lanjut:

  • Google Gemini API: Untuk fitur AI-powered hashtag generation. Data input Anda dikirim ke Google Gemini untuk diproses sesuai dengan Google Privacy Policy.
